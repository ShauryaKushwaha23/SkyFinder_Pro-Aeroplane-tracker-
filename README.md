# SkyFinder_Pro-Aeroplane-tracker-
SkyFinder Pro is a real-time aircraft radar web application built with Python and Streamlit that visualizes live global air traffic using the OpenSky Network API. It provides an interactive, map-based experience for tracking flights worldwide with live updates, filters, and analytics.
